# Hotel Booking Plugin

## Description
The Hotel Booking Plugin is a WordPress plugin that integrates with WooCommerce Bookings to provide a seamless hotel booking experience. It allows users to search for available hotels based on location and dates, view search results, and book their stay directly from the website.

## Features
- Search form for hotels with location, check-in, and check-out dates.
- Display of search results with available hotels.
- Integration with WooCommerce Bookings for managing bookings.
- Responsive design for a modern user experience.

## Installation
1. Upload the `hotel-booking-plugin` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[hotel_search_form]` to display the search form on any page or post.
4. Use the shortcode `[hotel_search_results]` to display the search results on any page or post.

## Usage
- To search for hotels, fill in the search form with your desired location and dates.
- After submitting the form, you will be redirected to the search results page displaying available hotels.
- Click on a hotel to view more details and complete your booking.
